# 실행
> 아래 경로를 참고하여, 제일 상단 hreview으로 이동해 주세요. <br>
> cmd 창에 **python manage.py runserver** 명령어를 입력해주세요. <br> 
> runserver 이후, url 경로에 **http://127.0.0.1:8000/hyuzi/** 를 입력하여 웹 서비스를 이용할 수 있습니다. <br>
